Using SplitCoin

SplitCoin is a decentralized application built with Ethereum. This means to use SplitCoin you'll need a connection to Web3. The recommended way to do this is to install Metamask. You can also use the Mist or some other Web3 enabled browser.

Shapeshift Integration

Shapeshift allows other currencies to benefit from SplitCoin. By using Shapeshift as a gateway to Ether, we can send the converted Ether to a SplitCoin contract. SplitCoin contracts can also send Ether to Shapeshift addresses. This allows us to achieve splits on multiple blockchains, as long as they are supported by Shapeshift.

Referrals

SplitCoin has a referral mechanism to encourage marketing. Users can generate referral contracts which receive a portion of the 0.25% dev fee. For each referral the referrer gets 10% of the dev fee on the contracts the referee creates.
